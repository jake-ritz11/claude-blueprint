# Example: _plans-config.md for a Python project

This shows how you might customize the Standards Files section of `_plans-config.md` for a Python project.

---

## Standards Files

Read the relevant standards files per the task type:

| File type        | Standards file                    |
| ---------------- | --------------------------------- |
| `*.py`           | `docs/python-style-guide.md`     |
| `test_*.py`      | `docs/testing-standards.md`      |
| `*.sql`          | `docs/sql-conventions.md`        |

Also check:

- `docs/ARCHITECTURE.md` for system design patterns
- `docs/ADR/` for architecture decision records
- `CONTRIBUTING.md` for contribution guidelines
